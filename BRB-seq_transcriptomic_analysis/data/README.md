# List of files used for the BRB seq transcriptomic data analysis pipeline :

- "STAR_SOLO_counts.xls"
File obtain using the [Nextflow RNAseq pipeline](https://forgemia.inra.fr/lpgp/rnaseq).

- orthologs.txt.gz

- "miRNA187_contextscore_output.txt"  
This file is the output table from tragetscan for the prediction of miR-187 targets.
The script used to identify conserved miRNA targets and nonconserved sites using a custom set of data is: [the Perl script](https://www.targetscan.org/vert_70/vert_70_data_download/targetscan_70.zip).
